📊CASO PROPUESTO
“Sistema de Gestión de Producción y Trazabilidad de mmpp Fresco para Exportación” 🌱

🌎 Contexto del negocio

La empresa AgroExport Verde Norte SAC es una agroindustrial peruana dedicada a la producción, procesamiento y exportación de espárrago fresco hacia mercados internacionales como Estados Unidos, Europa y Asia.

La empresa trabaja con fundos agrícolas propios y de terceros, desde donde se cosecha el espárrago fresco que posteriormente pasa por procesos de:

recepción
selección
lavado
clasificación
empaque
almacenamiento en frío
despacho para exportación

Debido a las exigencias sanitarias y fitosanitarias internacionales, la empresa necesita garantizar la trazabilidad completa del producto, desde el campo de cosecha hasta el cliente final.

Actualmente, gran parte de la información se registra manualmente en hojas de control y archivos dispersos, generando problemas como:

dificultad para rastrear lotes exportados
demoras en auditorías de calidad
poca visibilidad sobre el origen del producto
errores en el control de inventario
dificultad para identificar productos afectados ante incidencias sanitarias

Por ello, la empresa ha decidido desarrollar un Sistema de Gestión de Producción y Trazabilidad de Espárrago Fresco, que permita:

controlar la producción agrícola y de planta
gestionar lotes de cosecha y empaque
rastrear el producto desde el campo hasta el cliente
controlar calidad y condiciones sanitarias
gestionar inventarios y despachos
generar reportes para auditorías y certificaciones internacionales

🎯 Objetivo del sistema

Diseñar una base de datos que permita:

registrar información de campos agrícolas y cultivos
gestionar cosechas de espárrago
controlar lotes de producción y empaque
registrar procesos de selección y clasificación
gestionar controles de calidad
controlar inventarios en cámaras frigoríficas
registrar exportaciones y despachos
realizar trazabilidad completa del espárrago fresco

🧩 Alcance funcional

El sistema debe contemplar las siguientes áreas:

🌱 1. Gestión de Campos Agrícolas

La empresa administra distintos campos de cultivo donde se produce espárrago fresco.

Cada campo registra información como:

código de campo
nombre del fundo
ubicación
hectáreas cultivadas
variedad de espárrago
responsable agrícola

Cada campo puede tener múltiples zonas o parcelas de cultivo.

👨‍🌾 2. Gestión de Cosechas

La cosecha se realiza diariamente y debe registrarse detalladamente.

Cada cosecha registra:

fecha de cosecha
campo o parcela de origen
cantidad cosechada
turno de trabajo
cuadrilla responsable
operarios participantes

Cada cosecha genera uno o varios lotes de cosecha.

🏭 3. Recepción y Procesamiento en Planta

El espárrago cosechado es trasladado a planta para su procesamiento.

Durante el proceso se registran actividades como:

recepción de producto
lavado
selección
clasificación por calibre
descarte
empaque

Cada lote procesado registra:

fecha y hora de ingreso
peso recibido
peso procesado
porcentaje de merma
responsable del proceso

🏷️ 4. Gestión de Lotes y Trazabilidad

La empresa trabaja con lotes para garantizar la trazabilidad completa del producto.

Cada lote registra:

código de lote
fecha de procesamiento
parcela de origen
variedad de espárrago
calibre
tipo de empaque
peso total
estado del lote

La trazabilidad debe permitir:

🔍 Trazabilidad hacia atrás

Identificar:

de qué campo proviene el espárrago
qué parcela lo produjo
qué fecha de cosecha tuvo
qué operarios participaron
qué insumos agrícolas fueron utilizados
🔎 Trazabilidad hacia adelante

Identificar:

qué clientes recibieron determinado lote
qué contenedores transportaron el producto
qué lotes deben inmovilizarse ante alertas sanitarias
🧪 5. Control de Calidad

El sistema debe registrar inspecciones de calidad y control fitosanitario.

Cada control registra:

fecha de inspección
lote inspeccionado
temperatura del producto
calibre
color
firmeza
presencia de defectos
resultado de inspección
inspector responsable

Los resultados pueden clasificarse como:

aprobado
observado
rechazado

También se deben registrar:

incidencias sanitarias
tratamientos aplicados
observaciones técnicas

❄️ 6. Gestión de Inventario y Cámara Fría

Los productos empacados se almacenan en cámaras frigoríficas antes de su exportación.

Cada movimiento de inventario registra:

fecha
lote
cámara asignada
temperatura
cantidad almacenada
movimiento
ingreso
salida
traslado

El sistema debe permitir controlar:

stock por lote
tiempo de permanencia en frío
ubicación exacta del producto

🚢 7. Exportación y Despachos

Los lotes aprobados son preparados para exportación.

Cada despacho registra:

cliente internacional
país de destino
fecha de embarque
contenedor utilizado
lote exportado
cantidad enviada
naviera
puerto de salida

El sistema debe permitir identificar:

qué cliente recibió cada lote
qué contenedor transportó el producto
historial de exportación por lote

👥 8. Gestión de Clientes y Exportaciones

La empresa exporta a supermercados y distribuidores internacionales.

Cada cliente registra:

razón social
país
dirección
contacto comercial
certificaciones requeridas

Un cliente puede recibir múltiples despachos.

📈 Reglas de negocio clave

Un campo agrícola puede tener múltiples parcelas
Una parcela puede generar múltiples cosechas
Cada cosecha pertenece a una sola parcela
Una cosecha puede generar varios lotes de procesamiento
Un lote pertenece a una sola cosecha
Un lote puede pasar por múltiples controles de calidad
Un lote puede almacenarse en distintas cámaras frigoríficas
Un despacho puede incluir múltiples lotes
Un cliente puede recibir múltiples despachos
La trazabilidad debe permitir rastrear el espárrago desde la parcela hasta el cliente final
El control de inventario debe manejarse por lote y ubicación
Todo lote exportado debe contar con aprobación de calidad

exportaciones
